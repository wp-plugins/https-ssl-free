== Description ==

This plugin allows you to log into your wordpress using the free https service from medius project inc.

To more information go to http://mediusproject.org and http://intosecure.com/info-det.html

= What this Plugin does =

*or, a readme you **should** read before implementing this plugin on your site.*

This plugin filters the login and logout URLs that WordPress uses to generate its internal links.

== Installation ==

1. Upload https_ssl_free.php to the `/wp-content/plugins/` directory
